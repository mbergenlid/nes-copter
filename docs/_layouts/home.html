As a pet project during the first half of 2017, I wrote a NES emulator in Rust. The emulator is nothing fancy really, it only supports the very basic NES operations but it plays a few games like Donkey Kong, Ice Climber and Ice Hockey. The purpose of this project was mainly to learn Rust for which I needed to write something and emulators are cool, so why not? However, deep diving into NES internals spiked my interest for writing a game for the NES which I'm going to do now and I also intend to document the process here.

The game I'm going to create is a version of an old flash game I know as Copter. Some people might know it better as Flappy Bird but it's basically the same thing.
